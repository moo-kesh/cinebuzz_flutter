# Cinebuzz Flutter

A Flutter application that displays trending and now playing movies using the TMDB API. Features include:

- Browse trending movies
- View now playing movies
- Search for movies
- Bookmark favorite movies

## Getting Started

To run this app, you'll need a TMDB API key. Sign up at [TMDB](https://www.themoviedb.org/) and get your API key.

Add your API key to a configuration file or environment variable as needed.

Run the app with `flutter run`.
